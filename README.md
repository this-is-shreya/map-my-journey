## Features and Interfaces
1. Landing Page
  <img width="949" alt="image" src="https://user-images.githubusercontent.com/62089952/220309379-5c3eddd6-099f-45b5-8907-09b28c3bd4ec.png">

- The web application allows you to view your life's journey in the form a graph and keeps track of the things you achieved, explored and faced difficulties       with.
  <img width="945" alt="image" src="https://user-images.githubusercontent.com/62089952/220310104-461580ec-1706-48af-b050-a687682f4141.png">

- You can earn various badges based on your score. For adding your first event, you get a starter badge, if your score>=50, you get an achiever badge.
- And lastly, if your score>=100, you get a perseverance badge.
  
  <img width="943" alt="image" src="https://user-images.githubusercontent.com/62089952/220311006-2f0063f6-858a-4525-a2ce-8932d13e3ccd.png">

2. Login Page
  <img width="710" alt="image" src="https://user-images.githubusercontent.com/62089952/220311381-a76ed265-36b4-481d-b18b-b3e7ed30452a.png">

3. Signup Page
  <img width="532" alt="image" src="https://user-images.githubusercontent.com/62089952/220311547-f5f8fab0-d0b1-452f-9cbb-fd38f64926d5.png">

4. Home Page
  <img width="952" alt="image" src="https://user-images.githubusercontent.com/62089952/220311769-bd9ba8fe-eadf-492f-900d-7d71fa35f642.png">

5. Add event
  - Here, you can select one or multiple options under any category of your choice and describe the event. _(Describing the events is optional, but if an event is described, then choosing an option from the dropdown is mandatory)_
 <img width="503" alt="image" src="https://user-images.githubusercontent.com/62089952/220312659-16841f31-4dd9-48de-a28d-5b562884c407.png">

6. View Journey
  - On this page, the user can view her journey in the form of a graph and can also view the details regarding the various things she tried, achieved and faced           difficulties with in the past month/year.
  <img width="300" height="500" src="https://user-images.githubusercontent.com/62089952/220313292-82aedb3d-8264-42fe-9baf-35dd01d99e38.jpg">

7. Profile Page
  - In the profile page, the user can view all the badges she has earned so far.
    <img width="950" alt="image" src="https://user-images.githubusercontent.com/62089952/220312977-e7be0b03-5047-4e4a-ac75-88ebfe87dd0e.png">
    Initially, the score of the user is 0.
    - For for selecting any of the options under _Did you try something new?_, the score increments by 1.
    - For selecting _My biggest achievement so far_, the score gets incremented by 2.
    - And for selecting any of the options under _Faced any difficulties?_, the score gets decremented by 1.

## Instructions
1. <code>https://github.com/this-is-shreya/map-my-journey.git</code>
2. <code>cd map-my-journey</code>
3. <code>npm install</code>
4. <code>cd server </code>
5. <code>npm install</code>
6. Create a .env file for the client and server side of the application and add the required credentials.
7. To start the client side of the app, use <code> npm start </code>. For the server side of the app, use <code>node server.js </code>
